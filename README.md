# go-self-update

In order to test you can run the following code from terminal/cmd
```
git clone https://github.com/Erachter/go-self-update/
cd go-self-update
git checkout 1.0.0
go build go-self-update.go
./go-self-update 
```
It should update to version 2.0.0.
You can check that is has succesfully updated by running again:
```
./go-self-update 
```
